# AI Workflow

## Inputs

- Figma or screenshot reference
- Visible copy
- Interaction notes
- Accessibility defects to avoid
- Target stack and testing requirements

## Reviewable Outputs

- Component map
- Data model
- Token inventory
- Accessibility checklist
- Test plan
- Implementation notes

## Guardrails

- Do not copy generated or third-party widget markup.
- Do not use proprietary image URLs or imported vendor CSS.
- Explain non-obvious accessibility decisions.
- Validate with automated tests before treating the implementation as complete.
